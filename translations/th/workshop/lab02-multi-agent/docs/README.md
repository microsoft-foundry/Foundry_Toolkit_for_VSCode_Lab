# Lab 02 - การทำงานแบบหลายเอเจนต์: การประเมินความเหมาะสมระหว่างประวัติย่อกับงาน

## เส้นทางการเรียนรู้ทั้งหมด

เอกสารนี้จะแนะนำคุณในการสร้าง ทดสอบ และปรับใช้ **เวิร์กโฟลว์แบบหลายเอเจนต์** ที่ประเมินความเหมาะสมระหว่างประวัติย่อกับงานโดยใช้เอเจนต์เฉพาะทางสี่ตัวซึ่งถูกประสานงานผ่าน **WorkflowBuilder**

> **ข้อกำหนดเบื้องต้น:** ให้ทำ [Lab 01 - เอเจนต์เดียว](../../lab01-single-agent/README.md) ให้เสร็จก่อนเริ่ม Lab 02
