## ความปลอดภัย

Microsoft ให้ความสำคัญกับความปลอดภัยของผลิตภัณฑ์ซอฟต์แวร์และบริการของเราอย่างจริงจัง ซึ่งรวมถึงที่เก็บซอร์สโค้ดทั้งหมดในองค์กร GitHub ของเรา

**กรุณาอย่ารายงานช่องโหว่ด้านความปลอดภัยผ่านปัญหาสาธารณะใน GitHub**

สำหรับข้อมูลการรายงานความปลอดภัย สถานที่ ข้อมูลติดต่อ และนโยบาย กรุณาทบทวนคำแนะนำล่าสุดสำหรับที่เก็บ Microsoft ได้ที่ [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)
