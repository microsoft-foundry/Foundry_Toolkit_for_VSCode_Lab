## Sikkerhed

Microsoft tager sikkerheden for vores softwareprodukter og -tjenester alvorligt, hvilket
inkluderer alle kildekoderepositorier i vores GitHub-organisationer.

**Rapporter venligst ikke sikkerhedssårbarheder via offentlige GitHub issues.**

For oplysninger om sikkerhedsrapportering, lokaliteter, kontaktinformation og politikker,
gennemgå venligst den seneste vejledning for Microsoft-repositorier på
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
