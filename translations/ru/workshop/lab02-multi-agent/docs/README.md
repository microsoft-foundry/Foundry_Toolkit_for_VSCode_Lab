# Лабораторная работа 02 - Мультиагентный рабочий процесс: Резюме → Оценка соответствия вакансии

## Полный учебный путь

В этой документации вы пройдете процесс создания, тестирования и развертывания **мультиагентного рабочего процесса**, который оценивает соответствие резюме вакансии с использованием четырех специализированных агентов, организованных через **WorkflowBuilder**.

> **Предварительное условие:** Завершите [Лабораторную работу 01 - Один агент](../../lab01-single-agent/README.md) перед началом Лабораторной работы 02.
