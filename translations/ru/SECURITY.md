## Безопасность

Microsoft серьезно относится к безопасности наших программных продуктов и услуг, включая все репозитории исходного кода в наших организациях на GitHub.

**Пожалуйста, не сообщайте об уязвимостях безопасности через публичные вопросы на GitHub.**

Для получения информации о сообщении о безопасности, местах, контактных данных и политиках, пожалуйста, ознакомьтесь с последними рекомендациями для репозиториев Microsoft по адресу
