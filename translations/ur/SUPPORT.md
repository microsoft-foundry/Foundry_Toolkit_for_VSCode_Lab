# سپورٹ

## مسائل درج کرنے اور مدد حاصل کرنے کا طریقہ  

یہ پروجیکٹ GitHub Issues کو بگز اور خصوصیات کی درخواستوں کا پتہ لگانے کے لیے استعمال کرتا ہے۔ نئے مسائل درج کرنے سے پہلے موجودہ مسائل کو تلاش کریں تاکہ نقل سے بچا جا سکے۔ نئے مسائل کے لیے، اپنا بگ یا خصوصیت کی درخواست ایک نئے Issue کے طور پر درج کریں۔

اس پروجیکٹ کے استعمال کے بارے میں مدد اور سوالات کے لیے، براہ کرم **ریپوزیٹری کا مینٹینر: یہاں معلامات داخل کریں کہ ریپوزیٹری کے مالکان یا کمیونٹی سے مدد کے لیے کیسے رابطہ کریں۔ یہ کوئی Stack Overflow ٹیگ یا دوسرا چینل ہو سکتا ہے۔ آپ کہاں لوگوں کی مدد کریں گے؟**۔
