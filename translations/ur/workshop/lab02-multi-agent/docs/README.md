# لیب 02 - ملٹی ایجنٹ ورک فلو: ریزیومے → جاب فٹ ایویلیویٹر

## مکمل لرننگ پاتھ

یہ دستاویز آپ کو ایک **ملٹی ایجنٹ ورک فلو** بنانے، جانچنے، اور تعینات کرنے کے مراحل سے گزارتا ہے جو چار مخصوص ایجنٹس کے ذریعے ریزیومے اور جاب کی مطابقت کا اندازہ لگاتا ہے، جن کو **WorkflowBuilder** کے ذریعے منظم کیا گیا ہے۔

> **پری ریکوئیسٹ:** لیب 02 شروع کرنے سے پہلے [لیب 01 - سنگل ایجنٹ](../../lab01-single-agent/README.md) مکمل کریں۔
