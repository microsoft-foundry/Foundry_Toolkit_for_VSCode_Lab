# ஆய்வகம் 02 - பல முகவர் வேலைப்ப流: ரெசுமே → வேலை பொருத்தம் மதிப்பீட்டாளர்

## முழு கற்றல் பாதை

இந்த ஆவணம் உங்களுக்கு நான்கு சிறப்பு முகவர்களை கொண்டு **WorkflowBuilder** மூலம் ஒருங்கிணைக்கப்பட்ட, ரெசுமே-க்கு வேலை பொருத்தம் மதிப்பீடு செய்யும் **பல முகவர் வேலைப்ப流** உருவாக்குவதையும், சோதிப்பதையும், தளவமைப்பதையும் வழிநடத்துகிறது.

> **முந்தைய தேவை:** ஆய்வகம் 02 தொடங்குவதற்கு முன்பு [ஆய்வகம் 01 - ஒரே முகவர்](../../lab01-single-agent/README.md) முடிக்கவும்.
