# ஆதரவு

## பிரச்சனைகளை எப்படி தாக்கல் செய்வது மற்றும் உதவி பெறுவது  

இந்தத் திட்டம் பிழைகள் மற்றும் அம்ச கோரிக்கைகளை கண்காணிக்க GitHub Issues-ஐ பயன்படுத்துகிறது. புதிய பிரச்சனைகளை தாக்கல் செய்வதற்கு முன்பு உள்ள பிரச்சனைகளைத் தேடவும், கோப்புகள் மறுபடியும் உருவாக்கப்படாதிட. புதிய பிரச்சனைகளுக்கு உங்கள் பிழை அல்லது அம்ச கோரிக்கையை புதிய Issue ஆக தாக்கல் செய்யவும்.

இந்தத் திட்டத்தை பயன்படுத்துவதற்கான உதவி மற்றும் கேள்விகளுக்காக, **ரெப்போ பராமரிப்பாளர்: ரெப்போ உரிமையாளர்கள் அல்லது சமூகத்துடன் உதவிக்கு எவ்வாறு தொடர்புகொள்வது என்பதை இங்கேச் சேர்க்கவும். இது ஒரு Stack Overflow டேக் அல்லது மற்ற சேனல் இருக்கலாம். நீங்கள் எங்கே உதவும்?**.

## மைக்ரோசாஃப்ட் ஆதரவு கொள்கை
