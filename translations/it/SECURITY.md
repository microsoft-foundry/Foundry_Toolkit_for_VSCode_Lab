## Sicurezza

Microsoft prende sul serio la sicurezza dei nostri prodotti software e servizi, che
includono tutti i repository di codice sorgente nelle nostre organizzazioni GitHub.

**Si prega di non segnalare vulnerabilità di sicurezza tramite issue pubblici su GitHub.**

Per informazioni sulla segnalazione di sicurezza, posizioni, contatti e politiche,
si prega di consultare le linee guida più recenti per i repository Microsoft su
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
