Tento projekt používá GitHub Issues pro sledování chyb a požadavků na funkce. Před podáním nového problému, prosím, vyhledejte existující problémy, abyste předešli duplicitám. Pro nové problémy podávejte váš bug nebo požadavek na funkci jako nový Issue.
