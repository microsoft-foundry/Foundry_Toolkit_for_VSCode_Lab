## Ασφάλεια

Η Microsoft λαμβάνει σοβαρά υπόψη την ασφάλεια των προϊόντων λογισμικού και υπηρεσιών μας, που περιλαμβάνει όλα τα αποθετήρια πηγαίου κώδικα στους οργανισμούς μας στο GitHub.

**Παρακαλούμε μην αναφέρετε τρωτά σημεία ασφαλείας μέσω δημόσιων θεμάτων του GitHub.**

Για πληροφορίες αναφοράς ασφάλειας, τοποθεσίες, στοιχεία επικοινωνίας και πολιτικές, παρακαλούμε εξετάστε τις τελευταίες οδηγίες για τα αποθετήρια της Microsoft στη διεύθυνση [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
