# ਲੈਬ 02 - ਮਲਟੀ-ਏਜੰਟ ਵਰਕਫਲੋ: ਰਿਜਿਊਮੇ → ਨੌਕਰੀ ਫਿੱਟ ਮੁਲਾਂਕਣ

## ਪੂਰਾ ਸਿੱਖਣ ਦਾ ਰਸਤਾ

ਇਹ ਦਸਤਾਵੇਜ਼ ਤੁਹਾਨੂੰ ਚਾਰ ਵਿਸ਼ੇਸ਼ ਏਜੰਟਾਂ ਦੇ ਜ਼ਰੀਏ ਪ੍ਰਬੰਧਤ ਇੱਕ **ਮਲਟੀ-ਏਜੰਟ ਵਰਕਫਲੋ** ਬਣਾਉਣ, ਪਰਖਣ ਅਤੇ ਤैनਾਤ ਕਰਨ ਲਈ ਰਾਹ ਦਿਖਾਉਂਦਾ ਹੈ ਜੋ ਰਿਜਿਊਮੇ-ਨੌਕਰੀ ਫਿੱਟ ਦਾ ਮੁਲਾਂਕਣ ਕਰਦਾ ਹੈ।

> **ਪੂਰਵਾਪੇਖਾ:** ਲੈਬ 02 ਸ਼ੁਰੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ [Lab 01 - Single Agent](../../lab01-single-agent/README.md) ਪੂਰਾ ਕਰੋ।
