## സുരക്ഷ

മൈക്രോസോഫ്റ്റ് ഞങ്ങളുടെ സോഫ്‌ട്രേയ്‌സ് ഉൽപ്പന്നങ്ങളും സേവനങ്ങളും ഉൾപ്പെടെയുള്ള GitHub സംഘടനകളിലെ എല്ലാ സോഴ്‌സ് കോഡ് റിപോസിറ്ററികളും നിരഭീകരതയോടെ സംരക്ഷിക്കുന്നു.

**സുരക്ഷാ ദൗര്‍ബല്യങ്ങള്‍ പബ്ലിക് GitHub ഇഷ്യൂസ് വഴി റിപ്പോര്‍ട്ട് ചെയ്യരുത്.**

സുരക്ഷാ റിപ്പോര്‍ട്ട് സമര്‍പ്പണ വിവരങ്ങള്‍, സ്ഥിതികള്‍, സമ്പർക്ക വിവരങ്ങള്‍, നയങ്ങള്‍ എന്നിവയ്ക്ക്, ദയവായി മൈക്രോസോഫ്റ്റ് റിപോസിറ്ററികള്‍ക്കായുള്ള ഏറ്റവും പുതിയ മാർഗ്ഗനിർദ്ദേശം [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md) പരിശോധിക്കുക.
