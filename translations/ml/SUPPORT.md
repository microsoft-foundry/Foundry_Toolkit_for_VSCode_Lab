# പിന്തുണ

## പ്രശ്‌നങ്ങൾ ഫയൽ ചെയ്യുകയും സഹായം തേടുകയും ചെയ്യാനുള്ള വിധം  

ഈ പ്രോജക്ട് ബഗുകളും ഫീച്ചർ അഭ്യർത്ഥനകളും ട്രാക്ക് ചെയ്യാൻ GitHub Issues ഉപയോഗിക്കുന്നു. പുത്തനെ പ്രശ്‌നങ്ങൾ ഫയൽ ചെയ്യുന്നതിനു മുൻപ് നിലവിലുള്ള പ്രശ്‌നങ്ങൾ തിരയുക,കൊഴുപ്പ് ഒഴിവാക്കാൻ. പുതിയ പ്രശ്‌നങ്ങൾക്കായി, നിങ്ങളുടെ ബഗ് അല്ലെങ്കിൽ ഫീച്ചർ അഭ്യർത്ഥന ഒരു പുതിയ ഇഷ്യുവായി ഫയൽ ചെയ്യുക.

ഈ പ്രോജക്ട് ഉപയോഗിക്കുന്നതിനെക്കുറിച്ച് സഹായത്തിനും ചോദ്യങ്ങൾക്കും, ദയവായി **റിപ്പോ പരിപാലകൻ: സഹായത്തിനായി റിപോ ഉടമയോടോ കമ്മ്യൂണിറ്റിയോടോ എങ്ങനെ ചേരാം എന്നതിന്റെ നിർദ്ദേശങ്ങൾ ഇവിടെ ചേർക്കുക. അത് സ്റ്റാക്ക് ഓവർഫ്ലോ ടാഗോ മറ്റേതെങ്കിലും ചാനലോ ഉണ്ടായിരിക്കാം. നിങ്ങൾ എവിടെ ആളുകളെ സഹായിക്കുമെന്ന് വ്യക്തമാക്കുക**.
