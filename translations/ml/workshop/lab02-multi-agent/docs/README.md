# ലാബ് 02 - മൾട്ടി-എജന്റ് വർക്ക്ഫ്ലോ: റിസ്യൂം → ജോബ് ഫിറ്റ് മൂല്യനിർണ്ണയം

## പൂര്‍ണ പഠനപഥം

ഈ ഡോക്യുമെന്റേഷൻ നാലു പ്രത്യേക ഏജന്റുകൾ ഉപയോഗിച്ചും **WorkflowBuilder** വഴി ഓർക്കസ്ട്രേഷൻ ചെയ്തും റിസ്യൂം-ജോബ് ഫിറ്റ് മൂല്യനിർണ്ണയം നടത്തുന്ന ഒരു **മൾട്ടി-എജന്റ് വർക്ക്ഫ്ലോ** നിർമ്മിക്കുക, പരിശോധന നടത്തുക, വിനിയോഗിക്കുക എന്നിവയിൽ നിങ്ങളെ നടന്നു കൊണ്ട് പോകുന്നു.

> **അഗ്രിമൂർചിതം:** ലാബ് 02 തുടങ്ങുന്നതിനു മുൻപ് [ലാബ് 01 - സിങിൾ ഏജന്റ്](../../lab01-single-agent/README.md) പൂർത്തിയായിരിക്കണം.
