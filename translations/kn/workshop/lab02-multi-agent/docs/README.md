# ಪ್ರಯೋಗಶಾಲೆ 02 - ಬಹು-ಏಜೆಂಟ್ ಕಾರ್ಯಪ್ರವಾಹ: ರೆಜ್ಯೂಮೆ → ಉದ್ಯೋಗ ಹೊಂದಾಣಿಕೆ ಮೌಲ್ಯಮಾಪಕ

## ಸಂಪೂರ್ಣ ಅಧ್ಯಯನ ಮಾರ್ಗ

ಈ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ನಿಮಗೆ ನಾಲ್ಕು ತಜ್ಞ ಏಜೆಂಟ್‌ಗಳನ್ನು **WorkflowBuilder** ಮೂಲಕ ಸಂಯೋಜಿಸಿ, ರೆಜ್ಯೂಮೆ-ತೊ=", ಉದ್ಯೋಗ ಹೊಂದಾಣಿಕೆಯನ್ನು ಮೌಲ್ಯಮಾಪನ ಮಾಡುವ **ಬಹು-ಏಜೆಂಟ್ ಕಾರ್ಯಪ್ರವಾಹ**ವನ್ನು ನಿರ್ಮಿಸಲು, ಪರೀಕ್ಷಿಸಲು, ಮತ್ತು ನಿಯೋಜಿಸಲು ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ.

> **ಹೊಸದನ್ನು:** ಪ್ರಯೋಗಶಾಲೆ 02 ಆರಂಭಿಸುವ ಮೊದಲು [ಪ್ರಯೋಗಶಾಲೆ 01 - ಏಕ ಏಜೆಂಟ್](../../lab01-single-agent/README.md) ಪೂರ್ಣಗೊಳಿಸಿ.
