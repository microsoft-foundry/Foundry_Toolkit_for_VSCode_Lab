## Security

Microsoft သည် ကျွန်ုပ်တို့၏ ဆော့ဖ်ဝဲထုတ်ကုန်များနှင့် ဝန်ဆောင်မှုများ၏ လုံခြုံရေးကို အလေးထားပြီး၊ ၎င်းတွင် ကျွန်ုပ်တို့ GitHub အဖွဲ့အစည်းများရှိ အားလုံးသော source code repository များပါဝင်သည်။

**ကျေးဇူးပြု၍ လုံခြုံရေးအခွင့်အယှက်များကို ပြည်သူ့ GitHub issue များမှတဆင့် မသတင်းပို့ပါနှင့်။**

လုံခြုံရေးဆိုင်ရာ သတင်းပို့ခြင်းအချက်အလက်များ၊ တည်နေရာများ၊ ဆက်သွယ်ရန်အချက်အလက်များနှင့် မူဝါဒများအတွက် Microsoft repository များအတွက် နောက်ဆုံး ညွှန်ကြားချက်ကို [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md) တွင် ပြန်လည်သုံးသပ်ပါ။
