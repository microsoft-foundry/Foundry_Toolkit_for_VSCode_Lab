# Microsoft ဖွင့်လင်းရင်းမြစ် အပြုအမူ ကုဒ်

ဤပရောဂျက်သည် [Microsoft ဖွင့်လင်းရင်းမြစ် အပြုအမူ ကုဒ်](https://opensource.microsoft.com/codeofconduct/) ကို လက်ခံအသုံးပြုထားပါသည်။

ရင်းမြစ်များ-

- [Microsoft ဖွင့်လင်းရင်းမြစ် အပြုအမူ ကုဒ်](https://opensource.microsoft.com/codeofconduct/)
- [Microsoft အပြုအမူ ကုဒ် အကြောင်း မေးခွန်းများ](https://opensource.microsoft.com/codeofconduct/faq/)
- မေးမြန်းချက်များ သို့မဟုတ် ပြဿနာများရှိပါက [opencode@microsoft.com](mailto:opencode@microsoft.com) သို့ ဆက်သွယ်နိုင်ပါသည်။
- ဝန်ထမ်းများအနေဖြင့် [aka.ms/opensource/moderation-support](https://aka.ms/opensource/moderation-support) တွင် ဆက်သွယ်နိုင်ပါသည်။

**အဆိုပြုချက်**  
ဤစာရွက်စာတမ်းကို AI ဘာသာပြန်ဝန်ဆောင်မှုဖြစ်သည့် [Co-op Translator](https://github.com/Azure/co-op-translator) မှတဆင့် ဘာသာပြန်ထားခြင်း ဖြစ်ပါသည်။ တိကျမှန်ကန်မှုကို ကြိုးစားခဲ့သော်လည်း၊ အလိုအလျောက် ဘာသာပြန်ခြင်းများတွင် အမှားများ သို့မဟုတ် အမှန်တကယ် မဖြစ်နိုင်မှုများ ပါဝင်နိုင်ကြောင်း သတိပြုပါရန် လိုအပ်ပါသည်။ မူရင်းစာရွက်စာတမ်းသည် ၎င်း၏ ကိုယ်ပိုင်ဘာသာဖြင့် အတည်ပြုရမည့် မူနှင့် သာကောင်းစွာ ယူဆရမည့် အရင်းအမြစ် ဖြစ်သည်။ အရေးကြီးသောသတင်းအချက်အလက်များအတွက် သက်ဆိုင်ရာ ပရိုဖက်ရှင်နယ် လူသားဘာသာပြန်ခြင်းကို အကြံပြုပါသည်။ ဤဘာသာပြန်မှုကို အသုံးပြုခြင်းမှ ဖြစ်ပေါ်သော အနားယူမှုများ သို့မဟုတ် မမှန်ကန်သော အဓိပ္ပါယ်ဖွင့်ဆိုမှုများအတွက် ကျွန်ုပ်တို့ တာဝန်မခံပါ။
