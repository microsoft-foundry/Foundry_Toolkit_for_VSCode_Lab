## бһ”бһүбҹ’бһ бһ¶бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶

### бһҒбҹ’бһңбҹҮ бһ¬бһҸбһҳбҹ’бһӣбҹғ `.env` бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

бһҜбһҖбһҹбһ¶бһҡ `.env` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ `PersonalCareerCopilot/` (бһ“бҹ…бһӣбһҫбһҖбһҳбҹ’бһҡбһ·бһҸбһҠбһјбһ… `main.py`):

```
PersonalCareerCopilot/
в”ңв”Җв”Җ .env                  вҶҗ Must be here
в”ңв”Җв”Җ main.py
в”ңв”Җв”Җ agent.yaml
в”ңв”Җв”Җ Dockerfile
в””в”Җв”Җ requirements.txt
```

бһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһ…бҹҶбһ–бҹ„бҹҮ `.env`бҹ–

```env
PROJECT_ENDPOINT=https://<your-project-name>.services.ai.azure.com/api/projects/<your-project-id>
MODEL_DEPLOYMENT_NAME=gpt-4.1-mini
```

> **бһҡбһҖбһҹбһёбһҸбһ· PROJECT_ENDPOINT бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–**

## бһ”бһүбҹ’бһ бһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ MCP

### MCP бһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһ·бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣ

**бһӣбһҖбҹ’бһҒбһҺбҹҲ:** бһҖбһ¶бһҸ gap бһ”бһ„бҹ’бһ бһ¶бһү "No results returned from Microsoft Learn MCP" бһҡбһә "No direct Microsoft Learn results found"бҹ”

**бһҳбһјбһӣбһ бҹҒбһҸбһ»бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһң:**

1. **бһ”бһүбҹ’бһ бһ¶бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҠбһ¶бһү** - Endpoint MCP (`https://learn.microsoft.com/api/mcp`) бһҳбһ·бһ“бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”
   ```powershell
   # бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ
   Invoke-WebRequest -Uri "https://learn.microsoft.com/api/mcp" -Method POST -UseBasicParsing
   ```
 бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҸбҹ’бһҡбһЎбһ”бҹӢ `200`, endpoint бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”

2. **бһҹбҹҶбһҺбһҪбһҡбһ–бҹҒбһҖбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң** - бһҲбҹ’бһҳбҹ„бҹҮбһҮбҹҶбһ“бһ¶бһүбһ–бҹҒбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһңбҹӮбһ„ Microsoft Learnбҹ”
   - бһӮбһәбһҮбһ¶бһҡбһҝбһ„бһ’бһҳбҹ’бһҳбһҸбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹҶбһ“бһ¶бһүбһ–бһ·бһҹбҹҒбһҹбһҺбһ¶бһҹбҹӢбҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһ¶бһ“ URL бһ”бһҳбҹ’бһҡбһ»бһ„бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ”

3. **бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһҹбһҳбҹҗбһҷ MCP** - бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ Streamable HTTP бһ”бһ¶бһ“бһ•бһ»бһҸбһ–бҹҒбһӣбҹ”
   - бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһүбҹ’бһ…бҹҒбһүбһҹбҹҶбһҺбһҫбһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸбҹ” бһҹбһҳбҹҗбһҷ MCP бһҮбһ¶бһҡбһҷбҹҲбһ–бҹҒбһӣбһҒбҹ’бһӣбһё бһ“бһ·бһ„бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһҮбһҪбһ”бһ…бһҳбҹ’бһӣбһ„бһЎбһҫбһ„бһңбһ·бһүбҹ”

   | бһҖбҹҶбһ бһ»бһҹбһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» | бһҳбһјбһӣбһ бҹҒбһҸбһ» | бһҖбһ¶бһҡбһҮбһҪбһҹбһҮбһ»бһӣ |
   |--------------|-------|-----|
   | `ModuleNotFoundError` | `requirements.txt` бһҒбҹ’бһңбҹҮбһҖбһүбҹ’бһ…бһ”бҹӢ | бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһүбҹ’бһ…бһ”бҹӢ бһ бһҫбһҷбһҠбһ¶бһҖбҹӢбһҮбһ¶вҖӢбһҳбҹүбһјбһҢбһ»бһӣвҖӢбһЎбһҫбһ„бһңбһ·бһү |
   | `RuntimeError: Missing required environment variable` | бһҳбһ·бһ“бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ env vars бһ“бҹ… `agent.yaml` | бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһ•бҹ’бһ“бҹӮбһҖ `environment_variables` бһҖбҹ’бһ“бһ»бһ„ `agent.yaml` |
   | `azure.identity.CredentialUnavailableError` | бһҳбһ·бһ“бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ Managed Identity | Foundry бһҖбҹҶбһҺбһҸбҹӢбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· - бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһҠбһ¶бһҖбҹӢбһҸбһ¶бһҳбһ•бҹ’бһ“бҹӮбһҖбһ–бһ„бҹ’бһҡбһёбһҖ |
   | `OSError: port 8088 already in use` | Dockerfile бһ”бһ„бҹ’бһ бһ¶бһү port бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ¬бһ”бһүбҹ’бһ бһ¶бһ–бҹ’бһҡбһҳбһ•бҹ’бһҹбҹҶ port | бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ `EXPOSE 8088` бһҖбҹ’бһ“бһ»бһ„ Dockerfile бһ“бһ·бһ„ `CMD ["python", "main.py"]` |
   | Container бһ…бһ¶бһҖбһ…бҹҒбһүбһҖбһјбһҠ 1 | бһҖбҹҶбһ бһ»бһҹбһҳбһ·бһ“бһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ“бҹ… `main()` | бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһ‘бҹ’бһҡбһ„бҹӢбһҳбһ»бһ“ ([Module 5](05-test-locally.md)) бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбҹҶбһ бһ»бһҹ |

3. **бһҠбһ¶бһҖбҹӢбһ…бһјбһӣбһңбһ·бһүбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҮбһҪбһҹбһҮбһ»бһӣ:**
   - `Ctrl+Shift+P` вҶ’ **Microsoft Foundry: Deploy Hosted Agent** вҶ’ бһҮбҹ’бһҡбһҫбһҹбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ вҶ’ бһҠбһ¶бһҖбҹӢбһҮбһ¶бһңбһҫбһҹбһ·бһ“бһҗбҹ’бһҳбһёбҹ”

### бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҳбҹүбһјбһҢбһ»бһӣбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҷбһјбһҡ

Containers multi-agent бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҷбһјбһҡбһҮбһ¶бһ„ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸ instance бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бҹӨ бһ“бҹ…бһ–бҹҒбһӣбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбҹ” бһ–бҹҒбһӣбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ’бһҳбҹ’бһҳбһҸбһ¶бҹ–

| бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ | бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҡбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ |
|-------|------------------|
| бһ•бҹ’бһҹбҹҶбһҡбһјбһ”бһ—бһ¶бһ– container | бҹЎ-бҹЈ бһ“бһ¶бһ‘бһё |
| бһҠбһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ… ACR | бҹЈбҹ -бҹҰбҹ  бһңбһ·бһ“бһ¶бһ‘бһё |
| бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ container (single agent) | бҹЎбҹҘ-бҹЈбҹ  бһңбһ·бһ“бһ¶бһ‘бһё |
| бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ container (multi-agent) | бҹЈбҹ -бҹЎбҹўбҹ  бһңбһ·бһ“бһ¶бһ‘бһё |

> бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– "Pending" бһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһӣбһҫбһҹ бҹҘ бһ“бһ¶бһ‘бһё бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҖбҹҶбһ бһ»бһҹбһҖбҹ’бһ“бһ»бһ„ containerбҹ”

4. бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҒбҹ’бһңбҹҮ: бһ…бһ»бһ… **Add** вҶ’ **Add role assignment** вҶ’ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ **Azure AI User** вҶ’ бһ”бһ„бҹӢбһҮбһјбһ“бһӮбһҺбһ“бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһ”бһүбҹ’бһ бһ¶ Agent Inspector

### Inspector бһ”бһҫбһҖбһ”бһ¶бһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһ„бҹ’бһ бһ¶бһү "Disconnected"

2. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ port `5679`: Inspector бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҸбһ¶бһҳ debugpy бһ“бҹ… port 5679бҹ”

3. бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһЎбһҫбһ„бһңбһ·бһү бһ“бһ·бһ„бһ”бһҫбһҖ Inspector бһҠбһјбһ…бһҳбһ»бһ“бҹ”

### Inspector бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҶбһ¶бһ”бҹӢбһҸбһ·бһ…

бһ…бһҳбҹ’бһӣбһҫбһҷ multi-agent бһҳбһ¶бһ“бһ”бҹ’бһҡбһңбҹӮбһ„бһ’бҹҶ бһ“бһ·бһ„ stream бһҸбһ¶бһҳбһ”бһҺбҹ’бһҸбҹ„бһҷбҹ” бһҡбһ„бҹӢбһ…бһ¶бҹҶбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһӮбҹ’бһҡбһ”бҹӢбһӣбҹҶбһҠбһ¶бһ”бҹӢ (бһ”бҹ’бһҡбһ бҹӮбһӣ бҹЈбҹ -бҹҰбҹ  бһңбһ·бһ“бһ¶бһ‘бһё бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ…бҹҶбһ“бһҪбһ“ gap cards бһ“бһ·бһ„бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ MCP)бҹ”

бһ”бһҫбһӮбһ»бһҺбһ—бһ¶бһ–бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ…бҹ’бһҡбһҫбһ“бҹ–
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶ GapAnalyzer instructions бһҳбһ¶бһ“бһҒбҹ’бһ“бһ¶бһҸ `CRITICAL:` бһҠбҹӮбһӣбһҳбһ·бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ gap cardsбҹ”
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢ token бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҢбҹӮбһӣ - `gpt-4.1-mini` бһӮбһ¶бҹҶбһ‘бҹ’бһҡ token бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ бһјбһҸбһҠбһӣбҹӢ бҹЈбҹў,бҹ бҹ бҹ  бһҠбҹӮбһӣбһӮбһҪбһҡбһҸбҹӮбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбҹ”

## бһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бҹҶбһ–бһ„

## бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҮбҹҶбһ“бһҪбһҷ

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҒбһ¶бһ„бһӣбһҫбҹ–

1. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ** - бһҖбҹҶбһ бһ»бһҹбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһјбһң Python stack trace бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ terminalбҹ” бһўбһ¶бһ“ traceback бһ–бҹҒбһүбһӣбҹҒбһүбҹ”
2. **бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбһ¶бһҡбһҖбҹҶбһ бһ»бһҹ** - бһ…бһҳбҹ’бһӣбһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһҖбҹҶбһ бһ»бһҹ бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [Microsoft Q&A бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Azure AI](https://learn.microsoft.com/answers/tags/azure-ai-services)бҹ”
3. **бһ”бһҫбһҖбһ”бһүбҹ’бһ бһ¶** - бһ”бһүбҹ’бһҮбһјбһ“бһ”бһүбҹ’бһ бһ¶бһ“бҹ…бһӣбһҫ [workshop repository](https://github.com/ShivamGoyal03/ai-toolkit-hosted-agents-workshop/issues) бһҮбһ¶бһҳбһҪбһҷбҹ–
   - бһҹбһ¶бһҡбһҖбҹҶбһ бһ»бһҹ бһ¬ бһҡбһјбһ”бһ…бҹҶбһҺбһ¶бҹҶ
   - бһҖбҹҶбһҺбҹӮбһ”бһҺбҹ’бһҺpackage бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (`pip list | Select-String "agent-framework"`)
   - бһҖбҹҶбһҺбҹӮ Python бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (`python --version`)
   - бһҗбһ¶бһҸбһҫбһ”бһүбҹ’бһ бһ¶бһҮбһ¶бһ”бһҺбҹ’бһҠбһ¶бһӣбһҳбһҖбһ–бһёбһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ¬бһҖбҹ’бһҡбҹ„бһҷбһ–бҹҒбһӣбһҠбһ¶бһҖбҹӢбһ…бҹҒбһү
