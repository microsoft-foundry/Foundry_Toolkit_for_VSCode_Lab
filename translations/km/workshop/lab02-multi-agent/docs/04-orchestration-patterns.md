## бһӮбҹҶбһҡбһјбһ‘бһё 1бҹ– Fan-out (бһ…бҹӮбһҖбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһ—бһ¶бһ–)

бһӮбҹҶбһҡбһјбһҠбҹҶбһ”бһјбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбҹ„бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҮбһ¶ **fan-out** - бһ–бһҸбҹҢбһҳбһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҸбҹӮбһҳбһҪбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбҹ”

бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠ бһңбһ¶бһ“бһ№бһҖбһҹбҹ’бһҳбһ¶бһ“бһҗбһ¶ `resume_parser` бһӮбһәбһҮбһ¶бһўбҹ’бһ“бһҖбһ”бҹҶбһ–бҹҒбһүбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ - бһңбһ¶бһ‘бһ‘бһҪбһӣбһҹбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһ‘бһ¶бҹҶбһ„ `jd_agent` бһ“бһ·бһ„ `matching_agent` бһҳбһ¶бһ“бһўбһ¶бһҮбҹ’бһүбһ¶бһ’бһҡбһ–бһё `resume_parser` бһҷбҹүбһ¶бһ„бһҹбҹ„бһҸбһҹбҹ„бһҡбһҳбһҖбһңбһ·бһү бһ бҹ’бһңбҹ’бһҡбҹҒбһҳбһңбһҫбһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҠбһ№бһҖбһ“бһ¶бҹҶбһ•бһӣбһ”бһҸбҹӢбһ–бһё `resume_parser` бһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбҹ–

```python
.add_edge(resume_parser, jd_agent)         # бһӣбһ‘бҹ’бһ’бһ•бһӣ ResumeParser вҶ’ JD Agent
.add_edge(resume_parser, matching_agent)   # бһӣбһ‘бҹ’бһ’бһ•бһӣ ResumeParser вҶ’ MatchingAgent
```

**бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһңбһ¶бһҖбһҫбһҸбһҳбһ¶бһ“бҹ–** ResumeParser бһ“бһ·бһ„ JD Agent бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһ“бҹӮбһҖбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһ”бһҹбҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ” бһҖбһ¶бһҡбһҡбһҸбҹӢбһңбһ¶бһ–бһёбһҡбһ“бһ¶бһҖбҹӢбһҮбһ¶бһҹбһҳбһ—бһ¶бһ–бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҹбһҡбһ»бһ”бһҮбһ¶бһ„бһҖбһ¶бһҡбһҡбһҸбҹӢбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбҹ”

## бһӮбҹҶбһҡбһјбһ‘бһё 2бҹ– Fan-in (бһ”бһүбҹ’бһ…бһјбһӣбһ–бһ бһ»)

бһӮбҹҶбһҡбһјбһ‘бһёбһ–бһёбһҡ бһӮбһәбһҮбһ¶ **fan-in** - бһ•бһӣбһ”бһҸбҹӢбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣ бһ“бһ·бһ„бһ•бҹ’бһүбһҫбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҳбһҪбһҷбҹ”


**бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһҫбһҸбһҳбһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ–** бһ“бҹ…бһ–бҹҒбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһ¶бһ“ **бһ–бһёбһҡбһ¬бһ…бҹ’бһҡбһҫбһ“бһҖбҹҶбһ–бһјбһӣбһ”бһүбҹ’бһ…бһјбһӣ** бһ бҹ’бһңбҹ’бһҡбҹҒбһҳбһңбһҫбһҖбһ…бһ¶бҹҶбһ‘бһ»бһҖбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ **бһўбҹ’бһ“бһҖбһ”бҹҶбһ–бҹҒбһүбһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ¶бһӣбҹӢбһӮбҹ’бһ“бһ¶** бһҳбһ»бһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҡбһҸбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ” MatchingAgent бһҳбһ·бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһЎбһҫбһҷбһҡбһ бһјбһҸбһҠбһӣбҹӢ ResumeParser бһ“бһ·бһ„ JD Agent бһ”бһүбҹ’бһ…бһ”бҹӢбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбҹ”

### бһўбҹ’бһңбһёбһҠбҹӮбһӣ MatchingAgent бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“

бһ бҹ’бһңбҹ’бһҡбҹҒбһҳбһңбһҫбһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ•бһӣбһ”бһҸбҹӢбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһӣбһҫбһ”бһҺбҹ’бһҸбҹ„бһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһүбҹ’бһ…бһјбһӣбһҡбһ”бһҹбҹӢ MatchingAgent бһҳбһҫбһӣбһҠбһјбһ…бһҮбһ¶бҹ–

> **бһ…бҹҶбһҺбһ¶бҹҶбҹ–** бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҖбҹҶбһҺбҹӮбһ бҹ’бһңбҹ’бһҡбҹҒбһҳбһңбһҫбһҖбҹ” бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶбһҡбһ”бһҹбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһӮбҹ’бһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ”

## бһҖбһ¶бһҡбһўбһ¶бһ“бһҖбһјбһҠ WorkflowBuilder

бһ“бҹҒбҹҮбһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚ `create_workflow()` бһ–бҹҒбһүбһӣбҹҒбһүбһ–бһё `main.py` бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһҮбһҳбҹ’бһҡбһ¶бһ”бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ–

```python
def create_workflow(resume_parser, jd_agent, matching_agent, gap_analyzer):
    workflow = (
        WorkflowBuilder(
            name="ResumeJobFitEvaluator",

            # бһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„бһ‘бһёбһҳбһҪбһҷбһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ–бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
            start_executor=resume_parser,

            # бһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„бһҠбҹӮбһӣбһ•бһӣбһ”бҹҗбһҸбҹ’бһҡвҖӢбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ
            output_executors=[gap_analyzer],
        )
        # бһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ…бҹҒбһүбһ…бҹ’бһҡбһҫбһ“: бһӣбһ‘бҹ’бһ’бһ•бһӣ ResumeParser бһ•бҹ’бһүбһҫбһ‘бҹ…бһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„ JD бһ“бһ·бһ„ MatchingAgent бһ‘бһ¶бҹҶбһ„бһ–бһёбһҡ
        .add_edge(resume_parser, jd_agent)
        .add_edge(resume_parser, matching_agent)

        # бһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ…бһјбһӣбһ…бҹ’бһҡбһҫбһ“: MatchingAgent бһҡбһ„бҹӢбһ…бһ¶бҹҶбһ‘бһ¶бҹҶбһ„ ResumeParser бһ“бһ·бһ„ JD Agent
        .add_edge(jd_agent, matching_agent)

        # бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢ: бһӣбһ‘бҹ’бһ’бһ•бһӣ MatchingAgent бһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһ‘бҹ… GapAnalyzer
        .add_edge(matching_agent, gap_analyzer)

        .build()
    )
    return workflow.as_agent()
```

### бһҸбһ¶бһҡбһ¶бһ„бһҹбһ„бҹ’бһҒбҹҒбһ”бһ•бҹ’бһӣбһјбһңбһ…бҹ’бһҡбһҖ

| бһӣбҹҒбһҒ | бһ•бҹ’бһӣбһјбһңбһ”бһүбҹ’бһҮбһјбһ“ | бһӮбҹҶбһҡбһј | бһ–бһӣбһӣбһ‘бҹ’бһ’бһ•бһӣ |
|---|------|---------|--------|
| 1 | `resume_parser вҶ’ jd_agent` | Fan-out | JD Agent бһ‘бһ‘бһҪбһӣбһ•бһӣбһ”бһҸбҹӢбһ–бһё ResumeParser (бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһүбҹ’бһ…бһјбһӣбһҠбһҫбһҳ) |
| 2 | `resume_parser вҶ’ matching_agent` | Fan-out | MatchingAgent бһ‘бһ‘бһҪбһӣбһ•бһӣбһ”бһҸбҹӢбһ–бһё ResumeParser |
| 3 | `jd_agent вҶ’ matching_agent` | Fan-in | MatchingAgent бһҖбҹҸбһ‘бһ‘бһҪбһӣбһ•бһ„ JD Agent бһ•бһ„бһҠбҹӮбһҡ (бһҡбһ„бҹӢбһ…бһ¶бҹҶбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡ) |
| 4 | `matching_agent вҶ’ gap_analyzer` | бһ…бһ„бһҒбҹ’бһҹбҹӮ | GapAnalyzer бһ‘бһ‘бһҪбһӣбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһ“бҹғбһҖбһ¶бһҡбһҹбҹҶбһҡбһ»бһ„бһ“бһ·бһ„бһ”бһүбҹ’бһҮбһёбһ…бһ“бҹ’бһӣбҹ„бҹҮ |

## бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбҹ’бһҡбһ¶бһ”

### бһ”бһ“бҹ’бһҗбҹӮбһҳбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһҳбһҪбһҷ

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһёбһ”бҹ’бһҡбһ¶бҹҶ (бһ§. **InterviewPrepAgent** бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҶбһҺбһҪбһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһҹбҹ’бһўбһ¶бһҸбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ…бһ“бҹ’бһӣбҹ„бҹҮ)бҹ–

```python
# бҹЎ. бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ
INTERVIEW_PREP_INSTRUCTIONS = """\

# бҹў. бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһӣбһ»бһҖ async with)

    project_endpoint=PROJECT_ENDPOINT,

    credential=credential,

# бҹЈ. бһ”бһ“бҹ’бһҗбҹӮбһҳбһӮбҹӮбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ create_workflow()

.add_edge(gap_analyzer, interview_prep)     # бһҖбҹҸбһ‘бһ‘бһҪбһӣбһҖбһ¶бһҸбһҡбһ“бҹ’бһ’бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһҡбһ•бһ„бһҠбҹӮбһҡ

# бҹӨ. бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– output_executors
output_executors=[interview_prep],  # бһҘбһЎбһјбһңбһҮбһ¶ бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһўбһ“бһ»бһӮбҹ’бһҡбҹ„бҹҮбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ
```

### бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҡбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё

бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷ JD Agent бһҡбһҸбҹӢ **бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ** ResumeParser (бһҮбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢ бһҳбһ·бһ“бһҳбҹӮбһ“бһҹбһҳбһ—бһ¶бһ–)бҹ–

```python
# бһӣбһ»бһ”: .add_edge(resume_parser, jd_agent)  вҶҗ бһҳбһ¶бһ“бһҡбһҪбһ…бһ бһҫбһҷ, бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһңбһ¶
# бһӣбһ»бһ”бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһ„вҖӢбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһҖбһ¶бһҡбһҳбһ·бһ“бһІбҹ’бһҷ jd_agent бһ‘бһ‘бһҪбһӣбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ
# start_executor бһ•бҹ’бһүбһҫбһ‘бҹ… resume_parser бһҮбһ¶бһҳбһ»бһ“, бһ бһҫбһҷ jd_agent бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бһҸбҹҢбһҳбһ¶бһ“бһҸбҹӮ
# бһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһё resume_parser бһҸбһ¶бһҳбһҡбһҷбҹҲбһ…бҹҶбһҺбһ»бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ” бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбһ¶бһҮбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢбһӣбҹҶбһҠбҹ„бһҷбҹ”
```

> **бһҹбҹҶбһҺбһјбһҳбһ–бһҡ бҹ–** `start_executor` бһӮбһәбһҮбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбһҫбһҳбҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ‘бһ‘бһҪбһӣбһ•бһӣбһ”бһҸбҹӢбһ–бһёбһ•бҹ’бһӣбһјбһңбһ”бһүбҹ’бһ…бһјбһӣбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷбһ‘бһ‘бһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбһҫбһҳбһ•бһ„ бһҖбҹҸбһӮбһҪбһҡбһҸбҹӮбһҳбһ¶бһ“бһ•бҹ’бһӣбһјбһңбһ–бһё `start_executor` бһҳбһҖбһңбһ·бһүбҹ”

## бһҖбҹҶбһ бһ»бһҹбһҹбһүбҹ’бһҮбһ¶бһҸбһ·бһӮбҹ’бһҡбҹ„бһ„бһҖбҹ’бһҡбһ¶бһ”

| бһҖбҹҶбһ бһ»бһҹ | бһӣбһҖбҹ’бһҒбһҺбҹҲ | бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ |
|---------|---------|-----|
| бһҒбҹ’бһңбҹҮбһ•бҹ’бһӣбһјбһңбһ‘бҹ… `output_executors` | бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһҸбҹӢ бһ”бҹүбһ»бһ“бҹ’бһҠбҹӮбһ•бһӣбһ”бһҸбҹӢбһ‘бһ‘бҹҒ | бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҳбһ¶бһ“бһ•бҹ’бһӣбһјбһңбһ–бһё `start_executor` бһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„ `output_executors` |
| бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ—бһ¶бһ–бһңбһ„бҹӢбһңбһ·бһү | бһҖбһ¶бһҡбһҮбҹ’бһҡбһ»бһӣбһңбһ„бҹӢбһ¬бһ–бҹҒбһӣбһҡбһ„бҹӢбһ…бһ¶бҹҶбһҳбһ·бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ | бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһӮбҹ’бһҳбһ¶бһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҺбһ¶бһҳбһҪбһҷбһ•бҹ’бһҸбһӣбҹӢбһ‘бҹ…бһңбһ·бһүбһ‘бҹ…бһҳбһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһӣбһҫбһҖбҹҶбһ–бһјбһӣбһ‘бҹҒ |
| бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `output_executors` бһӮбҹ’бһҳбһ¶бһ“бһ•бҹ’бһӣбһјбһңбһ”бһүбҹ’бһ…бһјбһӣ | бһ•бһӣбһ”бһҸбҹӢбһ‘бһ‘бҹҒ | бһ”бһ“бҹ’бһҗбҹӮбһҳбһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ `add_edge(source, that_agent)` бһҳбһҪбһҷ |
| бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `output_executors` бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“ fan-in | бһ•бһӣбһ”бһҸбҹӢбһҳбһ¶бһ“бһҸбҹӮбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷ | бһ”бҹ’бһҡбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бһӣбһ”бһҸбҹӢбһҸбҹӮбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бҹ’бһҡбһҳбһјбһӣ, бһ¬бһ‘бһ‘бһҪбһӣбһ•бһӣбһ”бһҸбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ |
| бһҒбҹ’бһңбҹҮ `start_executor` | `ValueError` бһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ | бһҸбҹӮбһ„бһҸбҹӮбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ `start_executor` бһҖбҹ’бһ“бһ»бһ„ `WorkflowBuilder()` |
