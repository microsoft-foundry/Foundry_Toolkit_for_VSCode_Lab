бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҢбһ»бһӣбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ workflow бһҮбһ¶бһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ”бһҪбһ“-бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ… [Microsoft Foundry](https://learn.microsoft.com/azure/foundry/agents/concepts/hosted-agents) бһҖбҹ’бһ“бһ»бһ„бһ“бһ¶бһҳбһҮбһ¶ **Hosted Agent**бҹ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ– container Docker бһҳбһҪбһҷ бһ”бһ„бҹ’бһ бҹ„бҹҮбһңбһ¶бһ‘бҹ…бһҖбһ¶бһ“бҹӢ [Azure Container Registry (ACR)](https://learn.microsoft.com/azure/container-registry/container-registry-intro) бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹҶбһҺбҹӮ hosted agent бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [Foundry Agent Service](https://learn.microsoft.com/azure/foundry/agents/how-to/publish-agent)бҹ”

> **бһ…бҹҶбһҺбһ»бһ…бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ–бһё Lab 01**: бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ” Foundry бһӮбһ·бһҸ workflow бһ”бһҪбһ“-бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶ hosted agent бһ–бһёбһҡбһҮбһ¶бһ“бҹӢбһҳбһҪбһҷ - бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ container бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ•бҹ’бһ‘бҹғбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹ…бһҸбҹӮбһӮбһәбһ…бһ»бһ„бһ”бһ„бҹ’бһўбһҹбҹӢ `/responses` бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”

   - бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһҸбҹҒбһҹбҹ’бһҸбһ‘бһ¶бҹҶбһ„ 3 бһҖбҹ’бһ“бһ»бһ„ [Module 5](05-test-locally.md) бһ бһҫбһҷ workflow бһ•бһӣбһ·бһҸбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бҹҒбһүбһӣбҹҒбһүбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҸбһ…бһ“бҹ’бһӣбҹ„бҹҮ бһ“бһ·бһ„ URL Microsoft Learn бҹ”

2. **бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһё [Azure AI User](https://learn.microsoft.com/azure/foundry/concepts/rbac-foundry):**
   - бһ”бһ¶бһ“бһ…бһ¶бһҸбҹӢбһңбһ¶бһҷбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„ [Lab 01, Module 2](../../lab01-single-agent/docs/02-create-foundry-project.md)бҹ” бһҹбһјбһҳбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбҹ–
   - [Azure Portal](https://portal.azure.com) вҶ’ бһ”бҹ’бһҡбһ—бһ–бһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ вҶ’ **Access control (IAM)** вҶ’ **Role assignments** вҶ’ бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶ **[Azure AI User](https://aka.ms/foundry-ext-project-role)** бһҳбһ¶бһ“бһ”бһүбҹ’бһҮбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһёбһўбҹ’бһ“бһҖбҹ”

   - бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ Accounts бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҶбҹ’бһңбҹҒбһ„бһ“бҹғ VS Codeбҹ” бһҲбҹ’бһҳбҹ„бҹҮбһӮбһҺбһ“бһёбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһ”бһ¶бһ“бҹ”

### бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ multi-agent

- **бһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ”бһҪбһ“бһўбҹ’бһ“бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ container бһҳбһҪбһҷбҹ”** Foundry бһҳбһҫбһӣбһ”бҹ’бһҡбһ—бҹҒбһ‘ hosted agent бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҳбһҪбһҷбҹ” бһҖбҹ’бһҡбһ¶бһ” WorkflowBuilder бһҡбһҸбҹӢбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бҹ”
- **бһҖбһ¶бһҡбһ бҹ… MCP бһ‘бҹ…бһҖбҹ’бһҡбҹ…бҹ”** Container бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһўбҹҠбһёбһ“бһ’бһәбһҺбҹҒбһҸбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ‘бҹ… `https://learn.microsoft.com/api/mcp`бҹ” бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ”бһҹбҹӢ Foundry бһ•бҹ’бһҠбһӣбҹӢбһ“бһјбһңбһ“бҹҒбҹҮбһҮбһ¶вҖӢбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбҹ”
- **[Managed Identity](https://learn.microsoft.com/python/api/overview/azure/identity-readme#managed-identity-support).** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһңбҹҒбһҺ hosted, `get_credential()` бһҖбҹ’бһ“бһ»бһ„ `main.py` бһҸбҹ’бһҡбһЎбһ”бҹӢ `ManagedIdentityCredential()` (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ бһ–бҹ’бһҡбҹ„бҹҮ `MSI_ENDPOINT` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ)бҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”

## бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 4бҹ– бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ

1. бһ”бһҫбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ…бҹҶбһ  Microsoft Foundry (бһ…бһ»бһ…бһҡбһјбһ” Foundry бһ“бҹ…бһӣбһҫбһ•бҹ’бһ‘бһ¶бҹҶбһ„ Activity Bar)бҹ”

3. бһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһҳбһ¶бһ“ URL бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ“бһҪбһҷбһ•бһ„бһҠбҹӮбһҡбһҠбһјбһ…бһҮбһ¶ fallback бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ agent бһ“бһ№бһ„бһ“бҹ…бһҸбҹӮбһ•бһӣбһ·бһҸбһӣбһ‘бҹ’бһ’бһ•бһӣ (бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“ URL бһ”бһ“бҹ’бһҸбһ•бҹ’бһ‘бһ¶бһӣбҹӢ)бҹ”
