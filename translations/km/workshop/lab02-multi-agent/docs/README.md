# កម្មសិក្សា ០២ - ស្ទ្រីមហត្ថកម្មច្រើនភ្នាក់ងារ៖ អ្នកវាយតម្លៃការចូលសម្របន៍ពីប្រវត្តិរស្មីទៅការងារ

## ផ្លូវការសិក្សាសរុប

ឯកសារ​នេះ​នាំ​អ្នក​ភ្ជាប់​គ្នា​ក្នុង​ការ​សង់ ការផ្ទៀងផ្ទាត់ និងការបញ្ចេញ​ដំណើរការ **ស្ទ្រីមហត្ថកម្មច្រើនភ្នាក់ងារ** ដែលវាយតម្លៃការចូលសម្របន៍ពីប្រវត្តិរស្មីទៅការងារដោយប្រើភ្នាក់ងារពិសេសបួនប្រភេទគ្រប់គ្រងតាមរយៈ **WorkflowBuilder**។

> **ការបណ្តុះបណ្តាលមុន៖** សម្រាប់ការចាប់ផ្តើមកម្មសិក្សា ០២ សូមបញ្ចប់ [Lab 01 - Single Agent](../../lab01-single-agent/README.md) មុន។
