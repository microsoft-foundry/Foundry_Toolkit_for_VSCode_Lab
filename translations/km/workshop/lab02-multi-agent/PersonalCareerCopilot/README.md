## бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһҳбҹ’бһҡбҹ„бһ„

```
PersonalCareerCopilot/
в”ңв”Җв”Җ .env.example        вҶҗ Template for environment variables
в”ңв”Җв”Җ .env                вҶҗ Your credentials (git-ignored)
в”ңв”Җв”Җ agent.yaml          вҶҗ Hosted agent definition (name, resources, env vars)
в”ңв”Җв”Җ Dockerfile          вҶҗ Container image for Foundry deployment

в””в”Җв”Җ requirements.txt    вҶҗ Python dependencies
```

## бһҜбһҖбһҹбһ¶бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—

### `agent.yaml`

бһҖбҹҶбһҺбһҸбҹӢбһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹҒбһңбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Foundry Agent Serviceбҹ–
- `kind: hosted` - бһҡбһҸбҹӢбһҮбһ¶бһҸбһ·бһҒбһҳбҹ’бһҡбһҫбһҡ(container) бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„
- `protocols: [responses v1]` - бһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҶбһҺбһ»бһ…бһ…бҹҒбһү HTTP `/responses`
- `environment_variables` - `PROJECT_ENDPOINT` бһ“бһ·бһ„ `MODEL_DEPLOYMENT_NAME` бһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ–бҹҒбһӣбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ

### `main.py`

бһҳбһ¶бһ“бҹ–
- **бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„** - бһ”бҹ’бһҡбһ¶бҹҶбһ”бһҪбһ“бһўбһҗбҹҒбһҡ `*_INSTRUCTIONS`, бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„бһҡбҹҖбһ„бҹ—бһҒбҹ’бһӣбһҪбһ“
- **бһ§бһ”бһҖбһҡбһҺбҹҚ MCP** - `search_microsoft_learn_for_plan()` бһ бҹ…бһ‘бҹ… `https://learn.microsoft.com/api/mcp` бһҸбһ¶бһҳ Streamable HTTP
- **бһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„** - `create_agents()` бһҮбһ¶бһӣбҹҶбһ“бһ¶бҹҶбһ”бҹ’бһҡбһҫ `AzureAIAgentClient.as_agent()`
- **бһҖбҹ’бһҡбһ¶бһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ** - `create_workflow()` бһ”бҹ’бһҡбһҫ `WorkflowBuilder` бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„бһҮбһ¶бһҳбһҪбһҷбһӣбҹҶбһ“бһ¶бҹҶбһ”бһ„бҹ’бһ бҹ„бҹҮ/бһ”бһүбҹ’бһ…бһјбһӣ/бһҮбһ¶бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ
- **бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ** - `from_agent_framework(agent).run_async()` бһҸбҹ’бһҡбһ„бҹӢбһҡбһ“бҹ’бһ’ 8088
