# Module 2 - бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһ“бһ·бһ„бһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҳбҹүбһјбһҢбҹӮбһӣ

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҢбһ»бһӣбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ (бһ¬бһҮбҹ’бһҡбһҫбһҹ) бһӮбһҳбҹ’бһҡбҹ„бһ„ Microsoft Foundry бһҳбһҪбһҷ бһ бһҫбһҷбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбҹ” бһҡбһ¶бһӣбҹӢбһҮбҹҶбһ бһ¶бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһ…бҹҒбһүбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢ - бһҸбһ¶бһҳбһҠбһ¶бһ“бһңбһ¶вҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢбҹ”

> бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбҹүбһјбһҢбҹӮбһӣбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҡбһҪбһ…бһ бһҫбһҷ бһўбһ¶бһ…бһҡбҹҶбһӣбһ„бһ‘бҹ… [Module 3](03-create-hosted-agent.md) бҹ”

## бһҮбҹҶбһ бһ¶бһ“ 1бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһ–бһё VS Code

бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ Microsoft Foundry бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бһҖбһ…бҹҒбһүбһ–бһё VS Code бһ‘бҹҒбҹ”

1. бһ…бһ»бһ… `Ctrl+Shift+P` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖ **Command Palette**бҹ”
2. бһңбһ¶бһҷбҹ– **Microsoft Foundry: Create Project** бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһңбһ¶бҹ”
3. бһ”бһүбҹ’бһҮбһёбһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһ»бҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһ бһ¶бһү вҖ” бһҮбҹ’бһҡбһҫбһҹ **Azure subscription** бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ–бһёбһ”бһүбҹ’бһҮбһёбҹ”
4. бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһ“бһҫбһІбҹ’бһҷбһҮбҹ’бһҡбһҫбһҹбһ¬бһ”бһ„бҹ’бһҖбһҫбһҸ **resource group**бҹ–
   - бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҗбҹ’бһҳбһёбҹ– бһңбһ¶бһҷбһҲбҹ’бһҳбҹ„бҹҮбһҳбһҪбһҷ (бһ§. `rg-hosted-agents-workshop`) бһ бһҫбһҷбһ…бһ»бһ… Enterбҹ”
   - бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҪбһ…бһ бһҫбһҷбҹ– бһҮбҹ’бһҡбһҫбһҹбһңбһ¶бһ–бһёбһ”бһүбҹ’бһҮбһёбһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһ»бҹҮбҹ”
5. бһҮбҹ’бһҡбһҫбһҹ **бһҸбҹҶбһ”бһ“бҹӢ**бҹ” **бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ–** бһҮбҹ’бһҡбһҫбһҹбһҸбҹҶбһ”бһ“бҹӢбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбҹ” бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ [region availability](https://learn.microsoft.com/azure/foundry/agents/concepts/hosted-agents#region-availability) вҖ” бһҮбһҳбҹ’бһҡбһҫбһҹбһ’бһҳбҹ’бһҳбһҸбһ¶ бһӮбһә `East US`, `West US 2`, бһ¬ `Sweden Central`бҹ”
6. бһ”бһүбҹ’бһ…бһјбһӣ **бһҲбҹ’бһҳбҹ„бҹҮ** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry (бһ§. `workshop-agents`)бҹ”
7. бһ…бһ»бһ… Enter бһ бһҫбһҷбһҡбһ„бҹӢбһ…бһ¶бҹҶбһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҡбҹ„бһӮбҹ’бһӮбҹҚбһІбҹ’бһҷбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…бҹ”

> **бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһӮбһәбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣ 2-5 бһ“бһ¶бһ‘бһёбҹ”** бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҮбһ¶бһӣбһҫбһҖбһҒбһ¶бһ„бһҹбҹ’бһ–бҹғбһҶбҹ’бһңбҹҒбһ„бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҳбһҡбһ”бһҹбҹӢ VS Codeбҹ” бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ·бһ‘ VS Code бһ“бҹ…бһ–бҹҒбһӣбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһ‘бҹҒбҹ”

8. бһ–бҹҒбһӣбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҡбһҪбһ… бһҹбҹҠбһ¶бһҷбһ”бҹҠбһ¶ **Microsoft Foundry** бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһӮбһҳбҹ’бһҡбҹ„бһ„бһҗбҹ’бһҳбһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһҡбҹ„бһҳ **Resources**бҹ”
9. бһ…бһ»бһ…бһӣбһҫбһҲбҹ’бһҳбҹ„бҹҮбһӮбһҳбҹ’бһҡбҹ„бһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ„бҹ’бһҡбһёбһҖ бһ бһҫбһҷбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһңбһ¶бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһ“бҹӮбһҖбһҠбһјбһ…бһҮбһ¶ **Models + endpoints** бһ“бһ·бһ„ **Agents**бҹ”

### бһҮбһҳбҹ’бһҡбһҫбһҹбһҮбҹҶбһ“бһҪбһҹбҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҳбһҡбһҷбҹҲбһ§бһ”бһҖбһҡбһҺбҹҚ Foundry Portal

бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ–

1. бһ”бһҫбһҖ [https://ai.azure.com](https://ai.azure.com) бһ бһҫбһҷбһ…бһјбһӣбҹ”
2. бһ…бһ»бһ… **Create project** бһ“бҹ…бһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһҠбһҫбһҳбҹ”
3. бһ”бһүбҹ’бһ…бһјбһӣбһҲбҹ’бһҳбҹ„бҹҮбһӮбһҳбҹ’бһҡбҹ„бһ„ бһҮбҹ’бһҡбһҫбһҹ subscription, resource group, бһ“бһ·бһ„бһҸбҹҶбһ”бһ“бҹӢбҹ”
4. бһ…бһ»бһ… **Create** бһ бһҫбһҷбһҡбһ„бҹӢбһ…бһ¶бҹҶбһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶бҹ”
5. бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҪбһ… бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ… VS Code вҖ” бһӮбһҳбҹ’бһҡбҹ„бһ„бһӮбһҪбһҡбһҸбҹӮбһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ sidebar Foundry бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ’бҹ’бһңбһҫRefresh (бһ…бһ»бһ…бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ refresh)бҹ”

## бһҮбҹҶбһ бһ¶бһ“ 2бҹ– бһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҳбҹүбһјбһҢбҹӮбһӣ

[бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһҹбҹҒбһңбһ¶](https://learn.microsoft.com/azure/foundry/agents/concepts/hosted-agents) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҳбҹүбһјбһҢбҹӮбһӣ Azure OpenAI бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„ [бһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҳбһҪбһҷбһҘбһЎбһјбһңбһ“бҹҒбҹҮ](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/create-resource#deploy-a-model)бҹ”

1. бһ…бһ»бһ… `Ctrl+Shift+P` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖ **Command Palette**бҹ”
2. бһңбһ¶бһҷбҹ– **Microsoft Foundry: Open [Model Catalog](https://learn.microsoft.com/azure/ai-foundry/openai/concepts/models)** бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһңбһ¶бҹ”
3. бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ– Model Catalog бһ”бһҫбһҖбһ“бҹ… VS Codeбҹ” бһҡбһ»бһҖбһҡбһҖ бһ¬бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖ **gpt-4.1**бҹ”
4. бһ…бһ»бһ…бһӣбһҫбһҖбһ¶бһҸбһҳбҹүбһјбһҢбҹӮбһӣ **gpt-4.1** (бһ¬ `gpt-4.1-mini` бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҗбҹ’бһӣбҹғбһҗбҹ„бһҖ)бҹ”
5. бһ…бһ»бһ… **Deploy**бҹ”

![VS Code Model Catalog showing gpt-4.1 model card with Deploy button](../../../../../translated_images/km/02-model-catalog-gpt41.4a48eddc2c53e3cc.webp)
6. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бҹ–
   - **Deployment name**: бһ‘бһ»бһҖбһҠбһҠбҹӮбһӣ (бһ§. `gpt-4.1`) бһ¬бһ”бһүбҹ’бһ…бһјбһӣбһҲбҹ’бһҳбҹ„бҹҮбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ” **бһ…бһ„бһ…бһ¶бҹҶбһҲбҹ’бһҳбҹ„бҹҮбһ“бҹҒбҹҮ** вҖ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһңбһ¶бһ“бҹ… Module 4бҹ”
   - **Target**: бһҮбҹ’бһҡбһҫбһҹ **Deploy to Microsoft Foundry** бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ‘бһҫбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”
7. бһ…бһ»бһ… **Deploy** бһ бһҫбһҷбһҡбһ„бҹӢбһ…бһ¶бҹҶбһҖбһ¶бһҡбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢ (1-3 бһ“бһ¶бһ‘бһё)бҹ”
