## бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ

бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бһ„бҹ’бһ бҹ„бҹҮ бһҹбһјбһҳбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһңбһҸбҹ’бһҗбһ»бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ” бһҖбһ¶бһҡбһӣбһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһ”бҹӮбһ”бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ”бҹҶбһҺбһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһҝбһ„бһ“бҹғбһҖбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҮбһҪбһ”бһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһ”бҹҶбһ•бһ»бһҸбҹ”

1. **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ‘бҹ…бһ“бһ№бһ„бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ:**
   - бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһҸбҹҒбһҹбҹ’бһҸбһ‘бһ¶бҹҶбһ„ 4 бһҖбҹ’бһ“бһ»бһ„ [Module 5](05-test-locally.md) бһ бһҫбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

2. **бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһё [Azure AI User](https://learn.microsoft.com/azure/foundry/concepts/rbac-foundry#built-in-roles):**
   - бһҸбһҪбһ“бһ¶бһ‘бһёбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [Module 2, бһҮбҹҶбһ бһ¶бһ“ 3](02-create-foundry-project.md)бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ”бҹ’бһҡбһ¶бһҖбһҠ бһҹбһјбһҳбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҘбһЎбһјбһңбһ“бҹҒбҹҮбҹ–
   - Azure Portal вҶ’ бһҜбһҖбһҹбһ¶бһҡбһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ вҶ’ **Access control (IAM)** вҶ’ бһҸбһ¶бһ” **Role assignments** вҶ’ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҲбҹ’бһҳбҹ„бҹҮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ вҶ’ бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶ **Azure AI User** бһҳбһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбҹ”

3. **бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһ”бҹ’бһҡбһҫ Azure бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Code:**
   - бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ Accounts бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ-бһҶбҹ’бһңбҹҒбһ„бһ“бҹғ VS Codeбҹ” бһҲбҹ’бһҳбҹ„бҹҮбһӮбһҺбһ“бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһ”бһ„бҹ’бһ бһ¶бһүбҹ”

4. **(бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) Docker Desktop бһҖбҹҶбһ–бһ»бһ„бһҡбһҸбҹӢ:**
   - Docker бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбһҳбҹ’бһҡбһјбһңбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ Foundry бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“ бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһүбҹ’бһ…бһ”бҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһ”бһ„бҹ’бһ бҹ„бҹҮбҹ”
   - бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„ Docker бһҹбһјбһҳбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһңбһ¶бһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ– `docker info`

## бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 1: бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮ

бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ–бһёбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бҹ„бҹҮ - бһҸбҹӮбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бһ¶бҹҶбһұбҹ’бһҷбһҳбһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”

### бһҮбһҳбҹ’бһҡбһҫбһҹ A: бһ”бһ„бҹ’бһ бҹ„бҹҮбһ–бһё Agent Inspector (бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ)

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ debugger (F5) бһ“бһ·бһ„ Agent Inspector бһҖбҹҶбһ–бһ»бһ„бһ”бһҫбһҖбҹ–

1. бһҳбһҫбһӣ бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹҶбһ–бһјбһӣ-бһҹбҹ’бһҸбһ¶бҹҶ бһ“бҹғбһ•бҹ’бһ‘бһ¶бҹҶбһ„ Agent Inspectorбҹ”
2. бһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ **Deploy** (бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ–бһ–бһҖбһҳбһ¶бһ“бһ–бҹ’бһҡбһҪбһүбһҳбҹ’бһҒбһ¶бһ„бһӣбһҫ вҶ‘)бҹ”
3. бһңбһ¶бһҡбҹүбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһ“бһ№бһ„бһ”бһҫбһҖбҹ”

### бһҮбһҳбҹ’бһҡбһҫбһҹ B: бһ”бһ„бҹ’бһ бҹ„бҹҮбһ–бһё Command Palette

1. бһ…бһ»бһ… `Ctrl+Shift+P` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖ **Command Palette**бҹ”

3. бһңбһ¶бһҡбҹүбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһ“бһ№бһ„бһ”бһҫбһҖбҹ”

## бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 2: бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮ

бһңбһ¶бһҡбҹүбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһ“бһ№бһ„бһҠбһ№бһҖбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбһ¶бһҳбһ”бҹӮбһ”бһ”бһ‘бҹ” бһ”бҹҶбһ–бҹҒбһүбһҖбҹ’бһ“бһ»бһ„бһӮбҹ’бһҡбһ”бҹӢбһҖбһ¶бһҡбһҹбҹ’бһҡбһ„бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ–

### 2.1 бһҮбҹ’бһҡбһҫбһҹбһӮбһҳбҹ’бһҡбҹ„бһ„бһӮбҹ„бһӣбһҠбҹ…

### 2.2 бһҮбҹ’бһҡбһҫбһҹбһҜбһҖбһҹбһ¶бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҖбһүбҹ’бһ…бһ”бҹӢ

1. бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһ“бһҫбһұбҹ’бһҷбһҮбҹ’бһҡбһҫбһҹбһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣбһҡбһ”бһҹбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”
2. бһҮбҹ’бһҡбһҫбһҹ **`main.py`** (Python) - бһ“бҹҒбҹҮбһҮбһ¶бһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһңбһ¶бһҡбҹүбһ¶бһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”

### 2.3 бһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“

| бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ | бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ | бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ |
|---------|------------------|-------|
| **CPU** | `0.25` | бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бҹ” бһ”бһ„бҹ’бһҖбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ“бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ„бһ¶бһҡбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ |
| **Memory** | `0.5Gi` | бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶ |

бһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ“бһ№бһ„бһҸбһҳбҹ’бһӣбҹғбһҖбҹ’бһ“бһ»бһ„ `agent.yaml`бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһҷбһҖбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбһ”бһ¶бһ“бҹ”

## бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 3: бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ бһ“бһ·бһ„ бһ”бһ„бҹ’бһ бҹ„бҹҮ

1. бһңбһ¶бһҡбҹүбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһ„бҹ’бһҒбҹҒбһ”бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһҮбһ¶бһҳбһҪбһҷбҹ–
   - бһҲбҹ’бһҳбҹ„бҹҮбһӮбһҳбҹ’бһҡбҹ„бһ„бһӮбҹ„бһӣбһҠбҹ…
   - бһҲбҹ’бһҳбҹ„бҹҮбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһ–бһё `agent.yaml`)
   - бһҜбһҖбһҹбһ¶бһҡбһҖбһүбҹ’бһ…бһ”бҹӢ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“
2. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҹбһ„бҹ’бһҒбҹҒбһ” бһ“бһ·бһ„ бһ…бһ»бһ… **Confirm and Deploy** (бһ¬ **Deploy**)бҹ”
3. бһҸбһ¶бһҳбһҠбһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Codeбҹ”

### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһҫбһҸбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮ (бһҮбҹҶбһ бһ¶бһ“бһӣбҹҶбһҠбһ¶бһ”бҹӢ)

бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“бҹ” бһҸбһ¶бһҳбһҠбһ¶бһ“бһ•бҹ’бһ‘бһ¶бҹҶбһ„ **Output** бһҡбһ”бһҹбҹӢ VS Code (бһҮбҹ’бһҡбһҫбһҹ "Microsoft Foundry" бһ–бһёбһ”бһүбҹ’бһҮбһёбһ’бҹ’бһӣбһ¶бһҖбҹӢбһ‘бһ¶бһү) бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ¶бһҳбһҠбһ¶бһ“бҹ–

1. **Docker build** - VS Code бһҹбһ¶бһ„бһҹбһ„бҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһҖбһүбҹ’бһ…бһ”бҹӢ Docker бһ–бһё `Dockerfile` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҹбһ¶бһҡбһҮбһ¶бһ“бҹӢ Docker:
   ```
   Step 1/6 : FROM python:<version>-slim
   Step 2/6 : WORKDIR /app
   ...
   Successfully built abc123def456
   ```

2. **Docker push** - бһҡбһјбһ”бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҸбһ·бһҸбһ‘бҹ… **Azure Container Registry (ACR)** бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ“бҹҒбҹҮбһўбһ¶бһ…бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣ 1-3 бһ“бһ¶бһ‘бһёбһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһ бҹ„бҹҮбһҠбҹҶбһ”бһјбһ„ (бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһӣбһҫбһҹ 100MB)бҹ”

3. **Agent registration** - бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Foundry Agent бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бҹ„бҹҮ (бһ¬бһҮбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһ”бһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһ¶бһ“бһҳбһ¶бһ“бһҡбһҪбһ…бһ бһҫбһҷ)бҹ” бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ–бһё `agent.yaml` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбҹ”

4. **Container start** - бһҖбһүбҹ’бһ…бһ”бҹӢбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Foundryбҹ” бһңбҹҒбһ‘бһ·бһҖбһ¶бһ“бҹҒбҹҮбһ•бҹ’бһҠбһӣбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹ„бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ бһҫбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҶбһҺбһ»бһ…бһ…бҹҒбһү `/responses`бҹ”

> **бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһҠбҹҶбһ”бһјбһ„бһҷбһәбһҸбһҮбһ¶бһ„** (Docker бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ•бҹ’бһҸбһ·бһҸбһҮбһ¶бһ“бҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ)бҹ” бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһўбһ¶бһ…бһҡбһ бҹҗбһҹбһҮбһ¶бһ„бһҠбҹ„бһҷбһҹбһ¶бһҡ Docker бһ‘бһ‘бһҪбһӣбһҮбһ¶бһ“бҹӢбһҠбҹӮбһӣбһҳбһ·бһ“бһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбҹ”

## бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 4: бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бҹ„бҹҮ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһ”бһ„бҹ’бһ бҹ„бҹҮбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбҹ–

1. бһ”бһҫбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ **Microsoft Foundry** бһҠбҹ„бһҷбһ…бһ»бһ…бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ Foundry бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Bar бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бҹ”
2. бһ–бһ“бҹ’бһӣбһҝбһ“бһ•бҹ’бһ“бҹӮбһҖ **Hosted Agents (Preview)** бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
3. бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһғбһҫбһүбһҲбҹ’бһҳбҹ„бҹҮбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһ§. `ExecutiveAgent` бһ¬бһҲбҹ’бһҳбҹ„бҹҮбһ–бһё `agent.yaml`)бҹ”
4. **бһ…бһ»бһ…бһӣбһҫбһҲбҹ’бһҳбҹ„бҹҮбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ** бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ“бҹ’бһӣбһҝбһ“бһңбһ¶бҹ”
5. бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҳбһҪбһҷбһ¬бһ…бҹ’бһҡбһҫбһ“ **РІРөСҖСҒРёРё** (бһ§. `v1`)бҹ”
6. бһ…бһ»бһ…бһӣбһҫбһңбһҫбһҹбҹ’бһҷбһ»бһ“бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣ **Container Details**бҹ”
7. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹҠбһ·бһ…бһ‘бһё **Status**бҹ–

   | бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– | бһўбһҸбҹ’бһҗбһ“бҹҗбһҷ |
   |--------|---------|
   | **Started** бһ¬ **Running** | бһҖбһүбҹ’бһ…бһ”бҹӢбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ бһҫбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбҹҖбһҳбһҡбһҪбһ…бһ бһҫбһҷ |
   | **Pending** | бһҖбһүбҹ’бһ…бһ”бҹӢбһҖбҹҶбһ–бһ»бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (бһҡбһ„бҹӢбһ…бһ¶бҹҶ 30-60 бһңбһ·бһ“бһ¶бһ‘бһё) |
   | **Failed** | бһҖбһүбҹ’бһ…бһ”бҹӢбһ”бһҡбһ¶бһҮбҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» - бһҳбһҫбһӣбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ) |
