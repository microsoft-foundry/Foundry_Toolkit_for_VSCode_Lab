   - бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһҳбһ¶бһ“ вҶ’ бһ”бһ“бҹ’бһҸвҖӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бҹ”

> **бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ Owner/Contributor бһҳбһ·бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбҹ–** Azure RBAC бһҳбһ¶бһ“вҖӢбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһ–бһёбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘ - *бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„* бһ“бһ·бһ„ *бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ*бҹ” Owner бһ“бһ·бһ„ Contributor бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸвҖӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ (бһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“, бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ) бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡвҖӢбһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ `agents/write` бһҠбҹӮбһӣбһҮбһ¶бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбҹӮвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбһҪбһ“бһ¶бһ‘бһё `Azure AI User`, `Azure AI Developer`, бһ¬ `Azure AI Owner` бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһҹбһјбһҳбһҳбһҫбһӣ [бһҜбһҖбһҹбһ¶бһҡ Foundry RBAC](https://learn.microsoft.com/azure/foundry/concepts/rbac-foundry)бҹ”

### 1.2 бһҖбҹҶбһ бһ»бһҹ `AuthorizationFailed` бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“

**бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҠбһҫбһҳбҹ–** бһўбҹ’бһ“бһҖвҖӢбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһ”бһ„бҹ’бһҖбһҫбһҸ бһ¬бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ’бһ“бһ’бһ¶бһ“ Azure бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡвҖӢбһ‘бһјбһ‘бһ¶бһҸбҹӢ / бһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“бһ“бҹҒбҹҮбҹ”

**бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–**
1. бһҹбһјбһҳбһҹбҹ’бһ“бһҫбһ”бҹ’бһҡбһ’бһ¶бһ“бһҡбһҠбҹ’бһӢбһ”бһ¶бһӣбһҖбһ¶бһҡбһ‘бһјбһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһҠбһӣбҹӢбһҸбһҪбһ“бһ¶бһ‘бһё **Contributor** бһ“бҹ…бһӣбһҫбһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“бһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
2. бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ“бҹҒбҹҮ бһҹбһјбһҳбһҹбҹ’бһ“бһҫбһўбҹ„бһҷбһӮбһ¶бһҸбҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„ Foundry бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ•бҹ’бһҠбһӣбҹӢбһҸбһҪбһ“бһ¶бһ‘бһё **Azure AI User** бһ“бҹ…бһӣбһҫбһӮбһҳбҹ’бһҡбҹ„бһ„бҹ”

## 2. бһҖбҹҶбһ бһ»бһҹ Docker (бһҸбҹӮбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„ Docker)

4. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ terminal:

   - бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ **Use the WSL 2 based engine** бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҳбһҫбһӣбһғбһҫбһүбҹ”
   - бһ…бһ»бһ… **Apply & restart**бҹ”
   - бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ WSL 2 бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„, бһ”бһҫбһҖ PowerShell бһҮбһ¶ Administrator бһ бһҫбһҷбһҡбһҸбҹӢ `wsl --install` бһҡбһҪбһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһЎбһҫбһ„бһңбһ·бһүбҹ”
6. бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҗбҹ’бһҳбһёбҹ”

## 3. бһҖбҹҶбһ бһ»бһҹвҖӢбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ

### 3.1 [`DefaultAzureCredential`](https://learn.microsoft.com/azure/developer/python/sdk/authentication/credential-chains#defaultazurecredential-overview) бһҳбһ·бһ“бһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“вҖӢбһҹбҹ’бһӣбһ¶бһҖбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ”бһ¶бһ“

```
Error: DefaultAzureCredential failed to retrieve a token from the included credentials.
```

**бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҠбһҫбһҳбҹ–** бһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бһ–бһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҺбһ¶бһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһҹбҹӮ `DefaultAzureCredential` бһҳбһ¶бһ“бһҹбҹ’бһӣбһ¶бһҖбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ‘бҹҒбҹ”

**бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ - бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҮбҹҶбһ бһ¶бһ“бҹ–**

1. **бһ…бһјбһӣбһЎбһҫбһ„бһңбһ·бһүбһҸбһ¶бһҳ Azure CLI** (бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ’бһҳбҹ’бһҳбһҸбһ¶бһ”бҹҶбһ•бһ»бһҸ):
   ```bash
   az login
   ```
   бһ”бһ„бҹ’бһўбһҪбһ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ”бһҫбһҖбһЎбһҫбһ„бҹ” бһ…бһјбһӣ, бһҡбһҪбһ…бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ… VS Code бҹ”

2. **бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ‘бһјбһ‘бһ¶бһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ–**
   ```bash
   az account show --query "{name:name, id:id}" -o table
   ```
   бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ“бҹҒбҹҮбһҳбһ·бһ“бһҳбҹӮбһ“бһҖбһ¶бһҡбһ‘бһјбһ‘бһ¶бһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң:
   ```bash
   az account set --subscription "<your-subscription-id>"
   ```

3. **бһ…бһјбһӣбһЎбһҫбһ„бһңбһ·бһүбһҸбһ¶бһҳ VS Codeбҹ–**
   - бһ…бһ»бһ…бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ **Accounts** (бһҡбһјбһ”бһ”бһ»бһҡбһҹ) бһ“бҹ…бһҮбҹ’бһҡбһ»бһ„бһҖбҹ’бһҡбҹ„бһҳвҖӢбһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„вҖӢбһ“бҹғ VS Codeбҹ”
   - бһ…бһ»бһ…бһҲбҹ’бһҳбҹ„бҹҮбһӮбһҺбһ“бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ вҶ’ **Sign Out**бҹ”
   - бһ…бһ»бһ…бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ Accounts бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ вҶ’ **Sign in to Microsoft**бҹ”
   - бһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бһјбһӣбһҸбһ¶бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбҹ”

   - бһҖбҹҶбһҺбһҸбҹӢбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„ `.env` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

### 5.2 бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бҹҒбһүбһҖбҹ’бһҡбҹ„бһҷбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ

3. бһ“бҹ…бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸ бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ **Container Details** вҶ’ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһүбҹ’бһҮбһёбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» (Logs) бһ¬бһҸбҹҶбһҺбһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ”
