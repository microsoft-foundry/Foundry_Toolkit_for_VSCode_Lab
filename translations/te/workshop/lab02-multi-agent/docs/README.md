# ల్యాబ్ 02 - బహుళ-ఏజెంట్ వర్క్‌ఫ్లో: రెజ్యూమ్ → ఉద్యోగ అనుకూలత మೌಲ్యాంకనం

## పూర్తిగా నేర్చుకునే మార్గం

ఈ డాక్యుమెంటేషన్ మీరు బహుళ-ఏజెంట్ వర్క్‌ఫ్లోని నిర్మించడం, పరీక్షించడం, మరియు విడుదల చేయడాన్ని చూపిస్తుంది, ఇది నాలుగు ప్రత్యేక ఏజెంట్ల ద్వారా రెజ్యూమ్‌కి ఉద్యోగ అనుకూలతను **WorkflowBuilder** ద్వారా సమన్వయం చేస్తుంది.

> **అవకాశం:** ల్యాబ్ 02 ప్రారంభించే ముందు [ల్యాబ్ 01 - సింగిల్ ఏజెంట్](../../lab01-single-agent/README.md) పూర్తి చేయండి.
