## భద్రత

మైక్రోసాఫ్ట్ మా సాఫ్ట్‌వేర్ ఉత్పత్తులు మరియు సేవల భద్రతను గంభీరంగా తీసుకుంటుంది, ఇందులో మా GitHub సంస్థలలోని అన్ని మూల కోడ్ రిపాజిటరీలు ఉన్నాయి.

**దయచేసి భద్రతా లోపాల గురించి పబ్లిక్ GitHub Issues ద్వారా నివేదించవద్దు.**

భద్రతా నివేదిక సమాచారం, స్థలాలు, సంప్రదింపు సమాచారం, మరియు విధానాల కోసం, దయచేసి మైక్రోసాఫ్ట్ రిపాజిటరీల తాజా సూచనలను వీక్షించండి [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
