# Code de conduite pour les projets open source de Microsoft

Ce projet a adopté le [Code de conduite pour les projets open source de Microsoft](https://opensource.microsoft.com/codeofconduct/).

Ressources :

- [Code de conduite pour les projets open source de Microsoft](https://opensource.microsoft.com/codeofconduct/)
