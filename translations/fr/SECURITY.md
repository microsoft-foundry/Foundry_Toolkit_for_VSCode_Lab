## Sécurité

Microsoft prend la sécurité de nos produits logiciels et services très au sérieux, ce qui inclut tous les référentiels de code source dans nos organisations GitHub.

**Veuillez ne pas signaler les vulnérabilités de sécurité via des issues publiques sur GitHub.**

Pour des informations sur le signalement des problèmes de sécurité, les emplacements, les contacts et les politiques, veuillez consulter les dernières recommandations pour les référentiels Microsoft à l’adresse [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
