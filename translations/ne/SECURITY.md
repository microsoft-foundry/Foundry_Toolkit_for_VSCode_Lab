## सुरक्षा

माइक्रोसफ्टले हाम्रा सफ्टवेयर उत्पादनहरू र सेवाहरूको सुरक्षा गम्भीरतापूर्वक लिन्छ, जसमा हाम्रा GitHub संगठनहरूमा रहेका सबै स्रोत कोड रिपोजिटोरिहरू समावेश छन्।

**कृपया सार्वजनिक GitHub समस्याहरू मार्फत सुरक्षा कमजोरीहरू रिपोर्ट नगर्नुहोस्।**

सुरक्षा रिपोर्टिङ जानकारी, स्थानहरू, सम्पर्क जानकारी, र नीतिहरूको लागि,
कृपया माइक्रोसफ्ट रिपोजिटोरिहरूको लागि नवीनतम निर्देशनहरूको समीक्षा गर्नुहोस्:
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)।
