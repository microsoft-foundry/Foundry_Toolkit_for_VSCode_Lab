# ल्याब ०२ - बहु-एजेन्ट कार्यप्रवाह: रिजुमे → जागिर मिलान मूल्याङ्कनकर्ता

## पूर्ण सिकाइ मार्ग

यो दस्तावेजले तपाईंलाई चारवटा विशेष एजेन्टहरूद्वारा व्यवस्थापन गरिएको **बहु-एजेन्ट कार्यप्रवाह** निर्माण, परीक्षण र परिनियोजन गर्ने प्रक्रियामा मार्गदर्शन गर्दछ जुन रिजुमे-देखि-जागिर फिट मूल्याङ्कन गर्छ।

> **पूर्वआवश्यकता:** ल्याब ०२ सुरु गर्नु अघि [ल्याब ०१ - एकल एजेन्ट](../../lab01-single-agent/README.md) पूरा गर्नुहोस्।
