# Suporta

## Paano mag-file ng mga isyu at humingi ng tulong  

Gumagamit ang proyektong ito ng GitHub Issues para subaybayan ang mga bug at kahilingan para sa mga feature. Mangyaring hanapin muna ang umiiral na mga isyu bago mag-file ng bagong isyu upang maiwasan ang mga duplicate. Para sa mga bagong isyu, mag-file ng iyong bug o kahilingan para sa feature bilang isang bagong Isyu.

Para sa tulong at mga tanong tungkol sa paggamit ng proyektong ito, mangyaring **TAGAPANGALAGA NG REPO: ILAGAY DITO ANG MGA INSTRUKSYON PARA SA PAKIKIPAG-UGNAY SA MGA MAY-ARI NG REPO O KOMUNIDAD PARA SA TULONG. MAARING ITO AY STACK OVERFLOW TAG O IBA PANG CHANNEL. SAAN KA MAGTUTULUNGAN SA MGA TAO?**.

## Patakaran sa Suporta ng Microsoft  

Ang suporta para sa **PROYEKTO o PRODUKTO** ay limitado sa mga mapagkukunan na nakalista sa itaas.
