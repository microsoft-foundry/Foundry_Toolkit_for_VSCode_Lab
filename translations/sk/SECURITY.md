## Bezpečnosť

Microsoft berie bezpečnosť našich softvérových produktov a služieb vážne, vrátane všetkých zdrojových kódových úložísk v našich organizáciách GitHub.

**Prosím, nehláste bezpečnostné zraniteľnosti cez verejné GitHub issues.**

Pre informácie o hlásení bezpečnostných problémov, miesta, kontaktné údaje a pravidlá, prosím, prečítajte si najnovšie pokyny pre úložiská Microsoft na
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
