## पूर्ण शिक्षण मार्ग

ही दस्तऐवज तुम्हाला चार तज्ञ एजंट्सद्वारे संचालित **मल्टी-एजंट वर्कफ्लो** तयार करण्यास, चाचणी घेण्यास, आणि तैनात करण्यास मार्गदर्शन करते जे रिझ्युमे-टू-नोकरदार योग्यतेचे मूल्यांकन करते.

> **पूर्वशर्ती:** लॅब 02 सुरू करण्यापूर्वी [लॅब 01 - सिंगल एजंट](../../lab01-single-agent/README.md) पूर्ण करा.
