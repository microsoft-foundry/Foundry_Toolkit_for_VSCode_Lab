# समर्थन

## समस्या कशी नोंदवायची आणि मदत कशी मिळवायची  

हा प्रकल्प बग्स आणि वैशिष्ट्य अपीलांसाठी GitHub Issues वापरतो. नवीन समस्या नोंदविण्यापूर्वी विद्यमान समस्या शोधा ज्यामुळे डुप्लिकेट्स टाळता येतील. नवीन समस्या असल्यास, तुमचा बग किंवा वैशिष्ट्य अपील एका नवीन इश्यू म्हणून नोंदवा.

या प्रकल्पाचा वापर करण्याबाबत मदत आणि प्रश्नांसाठी, कृपया **रेपो देखभाल करणारा: येथे REPO मालक किंवा समुदायाकडे मदत घेण्यासाठी कशी संपर्क करायची यासाठी सूचना द्या. हे स्टॅक ओव्हरफ्लो टॅग अथवा अन्य चॅनेल असू शकते. तुम्ही लोकांना कुठे मदत करणार आहात?**.
