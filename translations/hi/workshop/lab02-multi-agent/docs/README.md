# लैब 02 - मल्टी-एजेंट वर्कफ़्लो: रिज़्यूमे → जॉब फिट मूल्यांकनकर्ता

## पूर्ण लर्निंग पाथ

यह दस्तावेज़ आपको एक **मल्टी-एजेंट वर्कफ़्लो** बनाने, परीक्षण करने और तैनात करने के बारे में मार्गदर्शन करता है जो चार विशेषज्ञ एजेंटों का उपयोग करके रिज़्यूमे-टू-जॉब फिट का मूल्यांकन करता है, जिन्हें **WorkflowBuilder** के माध्यम से समन्वित किया गया है।

> **पूर्वशर्त:** लैब 02 शुरू करने से पहले [लैब 01 - सिंगल एजेंट](../../lab01-single-agent/README.md) पूरा करें।
