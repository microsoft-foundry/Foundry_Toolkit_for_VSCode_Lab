# समर्थन

## कैसे मुद्दे दर्ज करें और सहायता प्राप्त करें  

यह प्रोजेक्ट बग और फीचर अनुरोधों को ट्रैक करने के लिए GitHub Issues का उपयोग करता है। कृपया नए मुद्दे दर्ज करने से पहले मौजूदा 
मुद्दों की खोज करें ताकि डुप्लीकेट से बचा जा सके। नए मुद्दों के लिए, अपना बग या 
फीचर अनुरोध एक नए Issue के रूप में दर्ज करें।

इस प्रोजेक्ट के उपयोग के बारे में सहायता और प्रश्नों के लिए, कृपया **रिपॉ मेंटेनर: यहां निर्देश डालें कि रिपॉ मालिकों या समुदाय से सहायता लेने के लिए कैसे जुड़ें। यह Stack Overflow टैग या कोई अन्य चैनल हो सकता है। आप लोगों की मदद कहां करेंगे?**।
