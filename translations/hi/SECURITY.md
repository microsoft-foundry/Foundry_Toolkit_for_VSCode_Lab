## सुरक्षा

Microsoft हमारे सॉफ़्टवेयर उत्पादों और सेवाओं की सुरक्षा को गंभीरता से लेता है, जिसमें हमारे GitHub संगठनों में सभी स्रोत कोड रिपॉजिटरी शामिल हैं।

**कृपया सार्वजनिक GitHub मुद्दों के माध्यम से सुरक्षा कमजोरियों की रिपोर्ट न करें।**

सुरक्षा रिपोर्टिंग जानकारी, स्थान, संपर्क जानकारी, और नीतियों के लिए, कृपया Microsoft रिपॉजिटरीज़ के लिए नवीनतम मार्गदर्शन की समीक्षा करें  
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md)।
