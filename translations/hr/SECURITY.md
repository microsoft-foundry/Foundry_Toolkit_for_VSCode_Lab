## Sigurnost

Microsoft ozbiljno shvaća sigurnost naših softverskih proizvoda i usluga, uključujući sve spremišta izvornog koda u našim GitHub organizacijama.

**Molimo vas da ne prijavljujete sigurnosne propuste putem javnih GitHub problema.**

Za informacije o prijavljivanju sigurnosnih problema, lokacije, kontakt podatke i politike, molimo pregledajte najnovije smjernice za Microsoft spremišta na
[https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
