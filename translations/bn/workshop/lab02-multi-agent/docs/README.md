# ল্যাব ০২ - মাল্টি-এজেন্ট ওয়ার্কফ্লো: রেজিউম → চাকরির সাথে ফিট মূল্যায়ক

## সম্পূর্ণ শেখার পথ

এই ডকুমেন্টেশন আপনাকে চারটি বিশেষায়িত এজেন্ট দ্বারা পরিচালিত **WorkflowBuilder** ব্যবহার করে রেজিউম থেকে চাকরির ফিট মূল্যায়ন করা **মাল্টি-এজেন্ট ওয়ার্কফ্লো** তৈরি, পরীক্ষা এবং মোতায়েন করার মাধ্যমে নিয়ে যাবে।

> **প্রয়োজনীয়তা:** ল্যাব ০২ শুরু করার আগে [ল্যাব ০১ - সিঙ্গল এজেন্ট](../../lab01-single-agent/README.md) সম্পূর্ণ করুন।
